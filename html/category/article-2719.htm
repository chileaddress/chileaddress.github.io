docker存储路径修改

docker修改对应的存储路径

修改daemon.json的内容不进行复述。

假设你已经编辑好/etc/docker/daemon.json文件的graph配置，要将其从d01修改到d02

{   "graph": "/d01/docker" }

则如果你需要修改他的存储位置，则首先停止docker服务，执行

service docker stop

创建d02

mkidr /d02/docker

再同步两个位置的数据，使用rsync、cp、mv均可

 rsync -avz /d01/docker/ /d02/docker

同步完成后，修改daemon.json

{  "graph": "/d02/docker" }

修改完成后，执行

service docker start

查看状态，如果没有异常则重启完成

systemctl status docker.service
