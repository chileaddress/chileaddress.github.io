vue引入js-base64

1.安装md5依赖

npm install --save js-base64

或者

cnpm install --save js-base64

2.在需要使用md5的组件中使用直接使用md5

let Base64 = require('js-base64').Base64;
Base64.decode('dsdsdSd');  Base64.decode('3232131');
Base64.encode('6545hgdfg');  Base64.encode('543hfghfhfhggfh');

使用Base64编码解码

1.下载包

npm install js-base64 -S  // 或者  cnpm install js-base64 -S

2.在你需要解码（编码）的文件中引用

let Base64 = require('js-base64').Base64

3.使用方法

// 解码  var a = Base64.decode()  // 编码  var b = Base64.encode()
